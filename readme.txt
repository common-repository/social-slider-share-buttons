== Installation ==
*For detailed description and installation details of the plugin visit plugin page at[Social Slider Share Buttons](http://www.wp-buy.com/).

*1.Download the package.
*2.Extract the contents of Social-Slider-Share-Buttons.zip to wp-content/plugins/ folder  You should get a folder called Social-Slider-Share-Buttons
*3.Activate the Plugin in WP-Admin.
*4.Goto Settings > Social-Slider-Share-Buttons > and then coose or configure options.
*5.You will find 3 optins, make changes to its own options to match your needs and save the changes
